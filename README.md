# POS System

Desktop Point of Sale app built with Electron + React + SQLite.

## Setup

```bash
cd pos-system
npm install
```

## Run (development)
Just th
```bash
npm run dev
```

## Build (production)

```bash
npm run build
```

## Features

- Product catalog with stock management
- Cart & checkout with change calculation
- Sales history with itemized receipts
- SQLite local database (no server needed)
