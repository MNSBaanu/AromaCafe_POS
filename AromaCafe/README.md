# AromaCafe

A warm, cozy cafe website built with React + Vite. Same architecture as FitZone, adapted for cafe energy.

## Stack
- React 19 + React Router v7
- Vite 8
- Pure CSS design system (no Tailwind)
- Scroll-triggered animations via Intersection Observer

## Design System
- Colors: espresso brown, warm amber, sage green, cream
- Fonts: Playfair Display (display/italic), Lato (UI), Merriweather (body)
- Dark theme with warm, cozy tones

## Pages
- Home, About, Menu, Baristas, Events, Blog, Contact, Loyalty
- Login / Register
- Role-based dashboards: Admin, Staff, Customer
